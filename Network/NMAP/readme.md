## NMAP

เมื่อเราดูไฟล์ log ของ nmap แล้ว เราจะพบว่ามันมีอะไรแปลกๆ
ใช่ครับ port ที่แสกนได้ไม่เรียงกัน และ มีซ้ำกันด้วย ทำให้ผมคิดว่านี่เป็นการซ่อน flag ด้วย cryptography แน่ๆ

ฉะนั้นเรามา list port กันเลยครับ

```

```
